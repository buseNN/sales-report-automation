# Automated Sales Report Generator

This project automates the creation of sales reports from Excel data using Python.

## Features
- Reads sales data from Excel file
- Calculates total and average sales
- Identifies top and worst performing products
- Generates a clean Excel report automatically

## Technologies
- Python
- Pandas
- OpenPyXL
- Excel

## How to use
1. Run create_data.py to generate sample data
2. Run sales_report.py to generate the report
3. Open daily_report.xlsx to see results

## Project Structure
- sales_report.py → main script
- create_data.py → generates sample data
- sales_data.xlsx → input data
- daily_report.xlsx → output report
